Dieser Ordner kann eins zu eins in den Ordner von der Arduino-IDE kopiert werden.
Hier sind unsere selbstgeschriebenen Header-Dateien.
Jede Header-Datei muss in einen einzelnen Ordner mit demselben Namen, damit die Arduino-IDE gut damit umgehen kann.

Informationen zur Installation von Bibliotheken finden Sie unter: http://www.arduino.cc/en/Guide/Libraries
